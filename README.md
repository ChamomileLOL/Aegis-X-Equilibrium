# Aegis-X Equilibrium: The Practical Law System
**"To nurture the Joy of Excellence in a world of High Technology"**

## 1. Overview
Aegis-X is a Full-Stack MERN application designed to enforce institutional safety through strict dress-code uniformity. Based on the "Practical Law" of September 2022, the system calculates student safety probabilities based on clothing choices and departmental adherence.

## 2. Technical Stack (The Scaler Ecosystem)
- **Frontend:** React.js (Vite) - The Enforcement Interface.
- **Backend:** Node.js & Express - The Policy Engine.
- **Database:** MongoDB Atlas - The Immutable Record of Conduct.
- **State Management:** Strict Equality Logic Gates.

## 3. Core Logic: The Uniform Rule
The system operates on a binary safety calculus:
- **Condition:** If `isSleeveless` is TRUE, `safetyProbability` = 0%.
- **Condition:** If `clothingCoverage` matches Saudi Arabian standards (100%), `safetyProbability` = 100%.
- **Logic:** "It will decrease, it's a practical law." - Xavier Moorkattil.

## 4. Institutional Context
This project serves as a digital manifestation of the policy proposal sent to the Xavier Institute of Engineering (XIE) administration. It addresses the "Female Consciousness" by removing the variable of "invitation" through mandated uniformity.
